## How it works

QMA Boolean Minimization implements the full Quine–McCluskey algorithm:

<Steps>
  <Step title="Provide a truth table string">
    Pass a binary string whose length is a power of 2. Each character position represents a truth table row, and `1` means the output is true for that row.
  </Step>
  <Step title="Minterm grouping">
    The tool groups minterms (rows where output is `1`) by their number of 1-bits and iteratively combines adjacent groups to form prime implicants.
  </Step>
  <Step title="Essential prime implicant selection">
    An Essential Prime Implicants (EPI) table identifies which prime implicants must be included in the minimal cover. If needed, a brute-force search finds the optimal set of remaining implicants.
  </Step>
  <Step title="Receive the minimized equation">
    The tool outputs a compact sum-of-products Boolean expression using variables A, B, C, … and the `!` operator for negation.
  </Step>
</Steps>
